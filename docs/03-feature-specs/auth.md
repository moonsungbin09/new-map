# 기능 명세 - 로그인/인증

## 1) 이 기능이 하는 일
- 사용자를 식별해서 "누가 어떤 장소를 등록했는지" 구분한다.
- MVP에서는 이메일 로그인만 지원한다.

## 2) 사용자 흐름
1. 사용자가 이메일과 비밀번호를 입력한다.
2. 앱이 서버에 로그인 요청을 보낸다.
3. 성공하면 홈 화면으로 이동하고, 실패하면 이유를 안내한다.

## 3) 예외 상황
- 비밀번호가 틀리면 "이메일 또는 비밀번호를 확인해 주세요" 메시지를 보여준다.
- 네트워크가 끊기면 "인터넷 연결을 확인해 주세요" 메시지를 보여준다.
- 서버 오류가 나면 잠시 후 다시 시도하도록 안내한다.
