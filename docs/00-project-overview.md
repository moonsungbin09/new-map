# 프로젝트 한눈에 보기

이 문서는 이 앱이 무엇인지 아주 쉽게 설명하는 문서입니다.

## 1) 이 앱은 무엇인가요?
- 네이버 지도처럼 지도를 기반으로 하되, 지하나 산처럼 기존 지도에 없는 공간도 누구나 쉽게 직접 만들고 공유할 수 있는 앱

## 2) 누가 쓰나요?
- 새로운 장소를 기록하고 다른 사람과 공유하고 싶은 일반 사용자

## 3) 어떤 문제를 해결하나요?
- 기존 지도에 없는 장소를 사용자가 직접 표시하기 어렵다는 문제

## 4) 배포는 어디에 하나요?
- Azure

- 추가한 장소를 간단 목록으로 보기
