# 개발 가이드 (처음 시작용)

이 문서는 "어떻게 개발을 시작하는지"를 설명합니다.

## 1) 준비물
- Android Studio
- JDK
- Git
- Azure 계정(백엔드 배포용)

## 2) 시작 순서
1. 저장소 받기
2. 앱 실행해 보기
3. 작은 기능 하나 수정해 보기

## 3) 작업 방식
- 큰 작업을 작은 작업으로 나눠서 진행
- 한 번에 한 가지 기능만 완료하기

## 4) Azure 관련 기본 원칙
- 앱(안드로이드)과 서버를 분리해서 생각한다.
- 앱은 사용자 화면과 입력을 담당한다.
- 서버는 Azure에 배포하고, 앱은 서버 API를 호출한다.
