# 화면 흐름 (어떤 순서로 보이는지)

## 1) 화면 목록
- 로그인 화면
- 홈 화면(지도 + 장소 목록)
- 장소 등록 화면
- 장소 상세 화면
- 설정 화면

## 2) 기본 이동 흐름
1. 로그인 성공 -> 홈 화면
2. 홈 화면에서 "새 장소 추가" -> 장소 등록 화면
3. 장소 등록 성공 -> 장소 상세 화면
4. 장소 상세 화면에서 뒤로가기 -> 홈 화면
5. 홈 화면에서 설정 아이콘 -> 설정 화면

## 3) 화면별 핵심 입력/행동

### 로그인 화면
- 입력: 이메일, 비밀번호
- 행동: 로그인 버튼

### 홈 화면
- 행동: 지도 이동, 핀 선택, 새 장소 추가 버튼 클릭

### 장소 등록 화면
- 입력: 장소 이름(필수)
- 입력: 좌표(현재 위치 자동 채우기 + 수동 보정 가능)
- 입력: 공간 유형(지상/지하/산, 필수)
- 입력: 설명(선택)
- 행동: 저장 버튼

### 장소 상세 화면
- 행동: 장소 정보 확인, 공유 버튼 클릭

### 설정 화면
- 행동: 알림 켜기/끄기, 로그아웃
