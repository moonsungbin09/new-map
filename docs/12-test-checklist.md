# 수동 테스트 체크리스트 (비개발자용)

아래 10개를 모두 통과하면 MVP 출시 가능 상태로 본다.

## 시나리오 1: 로그인 성공
- 준비: 가입된 이메일/비밀번호 준비
- 실행: 로그인 버튼 클릭
- 기대 결과: 홈 화면으로 이동

## 시나리오 2: 로그인 실패
- 준비: 틀린 비밀번호 준비
- 실행: 로그인 버튼 클릭
- 기대 결과: 오류 메시지 표시, 로그인 화면 유지

## 시나리오 3: 장소 등록 성공
- 준비: 장소 이름/좌표/공간 유형 입력
- 실행: 저장 버튼 클릭
- 기대 결과: 장소 상세 화면으로 이동

## 시나리오 4: 장소 등록 실패(필수값 누락)
- 준비: 장소 이름 비워두기
- 실행: 저장 버튼 클릭
- 기대 결과: "필수 항목을 입력해 주세요" 안내 표시

## 시나리오 5: 장소 목록 조회
- 준비: 장소 1개 이상 등록
- 실행: 홈 화면 목록 확인
- 기대 결과: 등록한 장소가 목록에 보임

## 시나리오 6: 장소 상세 조회
- 준비: 목록에 장소가 있음
- 실행: 장소 핀 또는 목록 아이템 클릭
- 기대 결과: 상세 정보 표시

## 시나리오 7: 장소 공유 링크 생성
- 준비: 장소 상세 화면 진입
- 실행: 공유 버튼 클릭
- 기대 결과: 공유 URL 생성

## 시나리오 8: 위치 권한 거부 시 동작
- 준비: 위치 권한 거부 상태
- 실행: 장소 등록 화면 진입
- 기대 결과: 앱이 종료되지 않고 수동 좌표 입력 가능

## 시나리오 9: 로그아웃
- 준비: 로그인 상태
- 실행: 설정 -> 로그아웃
- 기대 결과: 로그인 화면으로 이동

## 시나리오 10: 네트워크 끊김 예외
- 준비: 인터넷 연결 끄기
- 실행: 장소 목록 새로고침
- 기대 결과: 네트워크 오류 안내 표시
