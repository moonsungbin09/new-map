# 보안/개인정보 기준 (MVP)

## 1) 로그인/토큰 규칙
- 로그인 성공 시 서버가 `access_token`을 발급한다.
- 토큰 만료 시간은 기본 2시간으로 둔다.
- 로그아웃하면 앱의 토큰을 즉시 삭제하고, 서버에도 로그아웃 요청을 보낸다.

## 2) 권한 규칙
- 위치 권한은 사용자가 지도 기능에 처음 들어갈 때 요청한다.
- 사용자가 권한을 거부해도 앱은 멈추지 않고, 수동 좌표 입력으로 계속 진행할 수 있다.
- 권한 요청 문구는 쉬운 말로 안내한다.  
  예: "내 위치로 빠르게 장소를 찍기 위해 위치 권한이 필요해요."

## 3) 개인정보 최소 수집
- MVP에서 필수 개인정보는 이메일만 수집한다.
- 전화번호, 주소, 생년월일 등은 수집하지 않는다.
- 사용자 삭제 요청이 오면 다음 순서로 처리한다.
  1. 본인 확인
  2. 계정 및 관련 데이터 삭제
  3. 삭제 완료 안내
